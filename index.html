WCF-Proxy by steverb

Web proxy written in C# with WCF

This is a transparent web proxy written using C# and WCF. Currently it only supports "GET", but adding other verbs should be straight-forward. It will also translate urls in the proxied stream. It works similar to Apache's mod_proxy. Specify the URL you want to proxy in the web config along with the proxy service's address and you should be good to go.
